## 🚀 Step by Step Deploy ke GitHub Pages

### Step 1 — Buat Akun GitHub
Buka https://github.com → Sign up (gratis)

### Step 2 — Buat Repository Baru
1. Klik tombol "+" di pojok kanan atas
2. Pilih "New repository"
3. Isi nama repo, contoh: `baseposting-agent`
4. Pilih "Public"
5. Klik "Create repository"

### Step 3 — Upload File
1. Di halaman repo yang baru dibuat, klik "Add file"
2. Pilih "Upload files"
3. Drag & drop file `index.html` ke sana
4. Klik "Commit changes"

### Step 4 — Aktifkan GitHub Pages
1. Klik tab "Settings" di repo kamu
2. Scroll ke bawah, cari menu "Pages" di sidebar kiri
3. Di bagian "Source", pilih "Deploy from a branch"
4. Pilih branch "main", folder "/ (root)"
5. Klik "Save"

### Step 5 — Selesai!
Tunggu ~1 menit, lalu GitHub akan kasih link:
https://USERNAME.github.io/baseposting-agent

Share link itu ke siapa saja! 🎉
